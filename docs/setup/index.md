## 開発の進め方

開発はOpenAPIを出発点に、以下３つのステップで進めます。

1. UIドキュメントの作成
2. Figmaでデザイン作成
3. コード生成

一般にClaude CodeなどのAI Agent開発などでは、開発の初段階で厳格なプロンプト設計することで、アプリを最初から最後まで開発してもらいます。

一方本記事では、Github Copilotを使って、

1. フェーズの各単位における骨組みをAIで生成
2. それを人間が編集する

を繰り返してタスクを進めることで、より開発者の意図を反映しやすい形で開発を進めます。
